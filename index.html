1. Что у нас есть и как это работает

У нас настроено три инструмента обхода блокировок на сервере в Финляндии (Hetzner):

Инструмент Для чего Приложение
🔵 MTProto прокси Только для Telegram — быстро, стабильно Встроено в Telegram, приложение отдельно не нужно
🟢 VPN Amnezia Весь трафик устройства — сайты, приложения Amnezia VPN
🔴 VPN VLESS + Reality Весь трафик — устойчивый к блокировкам протокол v2rayNG (Android) / Streisand или FoXray (iPhone) / Hiddify (ПК)
💡 Рекомендация: если нужен только Telegram — используйте прокси. Для всего остального — Amnezia или VLESS.

2. 🔵 Прокси для Telegram

Работает прямо в приложении Telegram. Отдельно ничего скачивать не нужно.

✅ Самый быстрый способ — просто откройте ссылку ниже на устройстве, где установлен Telegram:
🔗 https://t.me/proxy?server=204.168.175.230&port=443&secret=ca23f090812c11c874f18c4e4348aa7c

Telegram откроется и предложит добавить прокси. Нажмите «Подключить» — готово.

Если ссылка не открывается — добавить вручную:

📱 iPhone (iOS)

1
Откройте Telegram → нажмите «Настройки» (нижний правый угол)
2
Выберите «Данные и память»
3
Прокрутите вниз → «Тип соединения»
4
Нажмите «Использовать прокси»
5
Нажмите «+» (добавить) → выберите «MTProto»
6
Заполните поля:
Сервер: 204.168.175.230
Порт: 443
Секрет: ca23f090812c11c874f18c4e4348aa7c
7
Нажмите «Сохранить»

📱 Android

1
Откройте Telegram → нажмите ☰ (три полоски) → «Настройки»
2
Выберите «Данные и память»
3
Прокрутите вниз → «Тип прокси»
4
Нажмите «Добавить прокси» → «MTProto»
5
Введите:
Сервер: 204.168.175.230
Порт: 443
Секрет: ca23f090812c11c874f18c4e4348aa7c
6
Нажмите «Сохранить» → «Подключить»

📱 Huawei (без Google)

Telegram можно скачать с AppGallery — он там есть официально. Дальше — те же шаги, как для Android выше.

💻 Windows / macOS / Linux

1
Откройте Telegram Desktop → «Настройки» (нижний левый угол)
2
«Конфиденциальность и безопасность»
3
Прокрутите вниз → «Тип соединения» → «Использовать прокси»
4
Нажмите «Добавить прокси» → MTProto и введите данные выше

3. 🟢 VPN Amnezia

Шифрует весь трафик устройства. Выглядит как обычный трафик, поэтому сложнее заблокировать.

Шаг 1: Скачайте приложение Amnezia VPN

📱 iPhone (iOS) — App Store

1
Откройте App Store → поищите «Amnezia VPN»
2
Или откройте прямую ссылку: apps.apple.com → AmneziaVPN
3
Нажмите «Загрузить» → установите

📱 Android — Google Play

1
Откройте Google Play → поищите «Amnezia VPN»
2
Или ссылка: play.google.com → AmneziaVPN
3
Нажмите «Установить»

📱 Huawei (без Google) — APK вручную

1
Откройте браузер → перейдите на сайт amnezia.org/download
2
Нажмите «Android APK» — скачается файл .apk
3
Откройте скачанный файл → разрешите установку из неизвестных источников (появится запрос) → установите

💻 Windows

1
Перейдите на amnezia.org/download
2
Нажмите «Windows» — скачается установщик .exe
3
Запустите файл → нажмите «Далее» → «Установить»

💻 macOS

1
Перейдите на amnezia.org/download
2
Нажмите «macOS» — скачается файл .dmg
3
Откройте .dmg → перетащите иконку Amnezia в папку Applications

💻 Linux

1
Перейдите на amnezia.org/download
2
Скачайте пакет для вашего дистрибутива (.AppImage, .deb или .rpm)
3
Для .AppImage: дайте права на исполнение и запустите. Для .deb: sudo dpkg -i файл.deb

5
В строке статуса должно появиться «Подключено»

4. 🔴 VPN VLESS + Reality

Самый устойчивый к блокировкам протокол. Подключение через QR-код.

💻 На компьютере QR обычно не сканируется — добавляйте профиль по ссылке (вручную) или через буфер обмена:
🔗 URL подписки (VLESS+Reality):
https://204.168.175.230:2096/sub/7fiw7aukux4kvu9u

📸 QR-код для подключения VLESS+Reality — отсканируйте приложением

📌 Для подключения нужно только отсканировать этот QR-код в соответствующем приложении. Никаких данных вводить не нужно.

📱 Android — приложение v2rayNG

Шаг 1: Скачать v2rayNG

1
Откройте Google Play → найдите «v2rayNG»
2
Или прямая ссылка: play.google.com → v2rayNG
3
Нажмите «Установить»

Шаг 2: Добавить сервер через QR

1
Откройте v2rayNG
2
Нажмите «+» (правый верхний угол)
3
Выберите «Импорт с QR-кода» (значок камеры)
4
Наведите камеру на QR-код вверху этой инструкции — сервер добавится автоматически
5
В списке серверов нажмите на добавленный, чтобы выделить его (галочка)
6
Нажмите кнопку ▶ (треугольник) внизу — VPN включится
7
Разрешите создание VPN-соединения при запросе

📱 Huawei (без Google) — v2rayNG через APK

1
Перейдите в браузере на github.com/2dust/v2rayNG/releases
2
Скачайте файл v2rayNG_x.x.x.apk (последняя версия)
3
Откройте скачанный .apk → разрешите установку → установите
4
Дальше — те же шаги, как для Android выше (добавить через QR)
